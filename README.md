hamletink/component
===================

> Common React components


Installation
------------

```
npm install --save @hamletink/component
```

Usage
-----

```js
import { StyleLoaderComponent } from '@hamletink/component';

// or

import StyleLoaderComponent from '@hamletink/component/style-loader';
```
